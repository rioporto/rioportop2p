INSTRUÇÕES PARA INICIAR O SERVIDOR:

1. Abra um novo terminal WSL (Ubuntu)

2. Execute estes comandos em sequência:

cd ~/Projetos/rioporto-app/rioporto-p2p
pkill -f "next dev"
rm -rf .next
npm run dev

3. Aguarde aparecer:
   ✓ Ready in XXXXms

4. Acesse no navegador:
   http://localhost:3000

SE DER ERRO DE PORTA EM USO:

cd ~/Projetos/rioporto-app/rioporto-p2p
PORT=3001 npm run dev

Depois acesse:
http://localhost:3001

IMPORTANTE: Mantenha o terminal aberto!
